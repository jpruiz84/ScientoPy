# ScientoPy v3.0.0 — Release Notes

## Highlights

ScientoPy 3.0.0 is a major release that modernizes the GUI framework from Tkinter to PySide6 (Qt 6), adds cross-platform distribution, and introduces a comprehensive test suite.

---

## New GUI Framework (PySide6)

- Complete rewrite of the GUI from Tkinter to **PySide6 (Qt 6)**
- **Automatic high-DPI scaling** — works natively on 4K/Retina displays without manual configuration
- **Resizable window** — minimum 853x480, freely resizable (replaces the old fixed-size window)
- **Light/Dark/System theme** — selectable via View > Appearance menu
- **Results tab** — shows the preprocess brief table directly inside the application after preprocessing
- **Extended Results tab** — shows detailed per-document analysis output with sortable columns
- **Column header tooltips** — hover over AGR, ADY, PDLY, h-index headers for definitions from the published paper
- **Widget tooltips** — all analysis controls show descriptive tooltips on hover
- **QSpinBox for year/numeric inputs** — built-in integer validation with up/down arrows (replaces free-text entry)
- **QSplitter** — draggable divider between the analysis form and custom topics area
- **Modal progress dialog** — uses QTimer-based polling instead of the old while/sleep loop
- **Native file dialogs** — Qt file/directory pickers replace Tkinter dialogs
- **Config persistence** — window geometry, splitter sizes, appearance mode, and last dataset folder saved to `scientopy_config.json`
- Matplotlib backend switched from TkAgg to **QtAgg** for consistent DPI and theme handling

## Cross-Platform Distribution

- New **PyInstaller spec** (`ScientoPyGui.spec`) replacing the Windows-only `winBuildScientoPyGui.spec`
- **GitHub Actions CI/CD workflow** for automated builds on Windows x64, Ubuntu x64, and macOS ARM64
- Asset path resolution fixed for PyInstaller bundles (logo and icon load correctly from `_internal/`)

## Data Processing Improvements

- **Open Access normalization** — standardizes OA tags between WoS (gold, hybrid, Bronze, Green Submitted/Accepted/Published) and Scopus (Gold/Hybrid Gold/Bronze/Green Open Access) formats
- **Open Access indicator extraction** from source datasets
- **Improved document type matching** logic for the `docType` criterion
- **PDLY rounded to 2 decimal places** in analysis output
- Removed unnecessary rounding in intermediate calculations

## Bug Fixes

- Fix `plot_evolution` crash with newer matplotlib versions (replace deprecated `ax.lines.append()` with `ax.add_line()`)
- Fix `set_window_title` error
- Fix escape sequences in BibTeX generation (`\&` and `\_`)
- BibTeX output path changed to `<inputfile>_bibliography.bib`

## Testing

- New **behavioral test suite** (73 tests) with synthetic data generator
- End-to-end tests covering: preprocessing, deduplication, analysis, all criteria, custom topics, trend analysis, previous results chaining, Open Access normalization

## Documentation

- Updated console manual to v3.0.0 with current install instructions and venv workflow
- Added BibTeX generation section to user manual and README
- Expanded manual with preprocessing steps, growth indicator formulas, and output column definitions
- Updated example dataset references to "Bluetooth low energy" (2026.04.02)
- Added `requirements.txt` for dependency management

## Dependencies

- **Added:** `PySide6>=6.6.0`
- **Removed:** `tkinter` (no longer imported)
- All other dependencies unchanged: numpy, pandas, scipy, matplotlib, Pillow, unidecode, wordcloud

## Breaking Changes

- The GUI entry point now requires PySide6 instead of Tkinter. Install via `pip install PySide6`
- Console scripts (`scientoPy.py`, `preProcess.py`) are **unchanged** and work exactly as before
