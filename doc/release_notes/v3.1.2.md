# ScientoPy v3.1.2 — Release Notes

## Highlights

This release adds Authenticode code signing for the Windows executable via the [SignPath Foundation](https://signpath.org) OSS program. The `ScientoPyGui.exe` shipped in the Windows release ZIP is now signed with a trusted certificate, replacing the unsigned binary from previous releases.

## Code Signing (Windows)

- **Signed Windows executable:** `ScientoPyGui.exe` inside `ScientoPy-windows-x64.zip` is now signed via the SignPath Foundation OSS program. Right-click the file → Properties → Digital Signatures to view the certificate chain.
- **Verified publisher:** Windows UAC and download dialogs will now display the verified publisher name instead of "Unknown publisher".
